Debugging in Keil

The following slide deck has instructions on debugging programs with the Keil µVision IDE. The pdf can be downloaded by clicking here.



If Keil µVision keeps crashing on pressing the Start\stop debug session, then follow these steps.

  1. Right click on the target folder (at the left pane).

  2. Select Options for Target `<TargetName>`.

  3. Select the Debug pane.

  4. Change the Parameter field to the right of Dialog DLL with text DP51.DLL to -p51.

  5. Tick the Limit Speed to Real-Time checkbox.
